Getting Started

the source files can be downloaded from github, using git or as a .zip file

Github repository 
git clone https://github.com/OysteinFalkeid/Game-Code

To run the code first install python 3 and run pip install from within the cloned repository

https://www.python.org/downloads/ 
pip install requirements.txt

This will install all required pacages

To run the program from terminal

python ./src/main.py

To build a .exe install and run pyinstaller 

pip install pyinstaller

Then run pyinstaler. On windows the aproriate paramaters for pyinstaller is

pyinstaller --name "Game-Code" --contents-directory "internal" --add-data=".\src\save":".\save" --add-data=".\src\sprites":".\sprites" --add-data=".\src\Consolas.ttf":"." ./src/main.py

This creates a folder "dist" containg the file "Game-Code.exe" and a "internal" folder containing all dependansies
